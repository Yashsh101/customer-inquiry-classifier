# Customer Inquiry Classifier

An end-to-end **NLP pipeline** to automatically classify customer queries into predefined categories like billing, technical support, shipping, and more.  
This project uses **Python, scikit-learn, and NLTK** to build and evaluate multiple machine learning models for text classification.

---

## 📋 Features
- Text preprocessing (tokenization, stopword removal, lemmatization)
- Multiple ML models (SVM, Logistic Regression, Naive Bayes, Random Forest)
- Model evaluation using accuracy, confusion matrix, and classification report
- Synthetic data generation for demonstration

---

## 🧠 Categories
- Billing  
- Technical Support  
- Product Inquiry  
- Shipping  
- Refund / Return  
- Account Management  
- General Inquiry
