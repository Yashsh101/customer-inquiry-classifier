# Customer Inquiry Classifier 🧠
This project classifies customer support inquiries into predefined categories using machine learning.

## Features
- Preprocesses text data
- Trains and evaluates multiple ML models
- Predicts inquiry category

## Tech Stack
- Python
- scikit-learn
- pandas
- numpy

## How to Run
```bash
pip install -r requirements.txt
python customer_inquiry_classifier.py
